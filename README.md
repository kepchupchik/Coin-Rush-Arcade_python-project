# 🎮 {Coin Rush Arcade} 

Динамичная 2D-аркада на Python, где игроку предстоит проявить скорость реакции, собирая монеты и избегая столкновений с врагами.

---

###  Геймплей и особенности

- **Система прогрессии**: Каждые 10 собранных монет повышают уровень персонажа, увеличивая сложность и темп игры.
- **Механики выживания**: 
  - Визуальная шкала здоровья (Health Bar) над персонажем.
  - Система временной неуязвимости (фреймы прозрачности) после получения урона.
- **Интуитивное управление**: Движение осуществляется с помощью стрелок на клавиатуре.
- **Стартовый экран**: Встроенное руководство пользователя прямо в игре перед началом раунда.

- **Архитектура**: Модульная структура (отдельные классы для игрока, уровней, предметов и настроек).

---

###  Структура файлов

- `main.py` — точка входа, управление состояниями игры (меню/геймплей).
- `playerr.py` — логика игрока: перемещение, расчет урона, анимация неуязвимости.
- `itemm.py` — класс предметов (монет) со случайной генерацией.
- `levell.py` — отрисовка уровней и интерфейса (UI).
- `settingss.py` — конфигурационный файл (FPS, цвета, размеры окна).
- `data/images/` — графические ресурсы (персонаж, враги, монеты).

###  Запуск игры

1. **Убедитесь, что установлен Python 3.x и Pygame:**
   ```bash
   pip install pygame```
2. **Клонируйте репозиторий:**
```
Bash
git clone [https://github.com/kepchupchik/game-project.git](https://github.com/kepchupchik/game-project.git)
cd game-project
```
3. **Запустите игру:**
```
Bash
python main.py
```
